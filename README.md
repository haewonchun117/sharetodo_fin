# ShareTodo (할일 공유 앱)

공통 분야의 관심사를 가진 유저들이 서로의 할 일을 공유하며 동기부여를 얻는 생산성 플랫폼입니다. 혼자 하는 할 일 관리에서 벗어나, 커뮤니티와 함께 목표를 달성해 보세요.

## 주요 기능
- 회원가입/로그인: 안전한 인증 시스템을 통해 나만의 계정을 생성하고 관리할 수 있습니다.
- 할 일 관리: 할 일을 직관적으로 추가하고 삭제하며 일상을 체계적으로 정리하세요.
- 카테고리별 공유: 공통 관심사(개발, 운동, 공부 등)를 선택하여 같은 분야의 다른 사람들이 어떤 목표를 가지고 있는지 확인하고 동기부여를 받을 수 있습니다.
- 다크 모드 지원: 사용자의 눈을 편안하게 해주는 세련된 다크 모드 인터페이스를 제공합니다.
- 사용자 친화적 디자인: 깔끔하고 세련된 UI로 누구나 손쉽게 사용할 수 있습니다.

## 기술 스택
- Framework: Next.js
- Backend/DB: Supabase
- Deployment: Vercel
- IDE: VS Code
- Version Control: GitHub
